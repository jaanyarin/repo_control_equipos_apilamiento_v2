## 6. Riesgos

| Riesgo | Impacto | Mitigación |
|---|---|---|
| EAS Cloud build falla por cambios nuevos | Alto | Verificar config con `npx expo config` pre-build |
| Firebase google-services.json sensible en repo | Medio | Agregar a `.gitignore`, documentar setup en `.env.example` |
| Tests lentos por Testcontainers PostgreSQL | Medio | Usar perfil con H2 para tests unitarios, Testcontainers solo para integración |
| CI/CD consume tiempo de ajuste fino | Medio | Usar workflow mínimo al inicio (build only) |
