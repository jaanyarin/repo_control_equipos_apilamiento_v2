# 4. Decisiones Técnicas

| Decisión | Detalle |
|---|---|
| FK como Long | ProveedorId, MarcaId, TipoEquipoId como columnas Long (no @ManyToOne) |
| Código auto-generado | Basado en nombre/modelo + serie, uppercase, snake_case |
| Soft-delete | `estadoActivo` booleano en todas las tablas operativas |
| Estado avería | `REPORTADA` → `ATENDIDA` con cálculo automático de días inactividad |
| Navegación mobile | NativeStack + BottomTab combinados |
| Formularios mobile | React Hook Form + Zod con Controller |
| Sin Lombok | Getters/setters manuales consistentes con HDT-001 |
