## 3. Decisiones Técnicas Adoptadas

- Bottom Tabs con 3 secciones: Inicio, Equipos, Perfil usando `@react-navigation/bottom-tabs`
- Formularios mobile con `react-hook-form` + `zod` + `@hookform/resolvers`
- Iconos en tabs con `react-native-vector-icons`
- Backend completo con las 6 capas (Entity, Repository, Service, Controller, DTO, Mapper)
- Frontend web CRUD con el mismo patrón que Sedes/Campañas existentes
